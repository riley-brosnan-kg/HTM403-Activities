User Guide

Welcome to the Financial Management Activity. Here's how to get started:

  1. Use the dropdown above to select "Instructions". Read through all instructions prior to starting the activity.
  2. Use the dropdown above to select "Interactive EBERS Activity".
  3. Navigate to EBERS. You will be pulling data from your own facility for this activity.
  4. Complete all required fields in each lab. Open-text fields are resizable using the bottom-right corner.
  5. Ask questions! Your instructors are here to support you.

Instructions

Before beginning the Interactive EBERS Activity, please read the following instructions carefully:

  • Make sure you have access to EBERS and any necessary login credentials.
  • You will be working with data from your own facility, so have that information ready.
  • The activity is divided into multiple sections—be sure to complete each one fully.
  • Text areas can be resized by dragging the bottom-right corner.
  • Click "Save" or "Submit" (if available) before navigating away from a section.
  • If you encounter issues, contact your instructor or support staff promptly.

Interactive EBERS Activity

This is the main activity section. Follow the prompts and input data from your facility into the fields provided.

  • Access EBERS and locate the necessary financial data for your site.
  • Enter the data in the designated fields below each question or prompt.
  • Use real, current data whenever possible to enhance learning and application.
  • Some fields will require text input; others may ask for specific numerical data or analysis.
  • If a section is unclear, refer back to the Instructions or ask your instructor for clarification.
